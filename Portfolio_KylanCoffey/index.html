Unit 3

In css (Cascading Style Sheet) I learned how to change the design and the layout of content in the webpages. I also learnd how to apply the style to the other webpages using a simple command. Then you connect them to a link code like: link type="text/css" rel="stylesheet" href="styles/mystyle.css". This helps so we dont have to copy and paste each style codes to each webpage. Css helps the look of the content and design of webpages to make sure it looks pleasing to the eye and for easy navigation.

Code that was helpful in Unit 3

  • How to change what the font of the text looked like using this:
    (font-family: Arial, Arial Black;)
  • To change the backround colors of things using this:
    (background-color: #4D4D4D; /* gray */)
  • To link the look of the css to each page without copying and pasting a bunch of code you paste this code in each page you want the design to be use this code:
    (link type="text/css" rel="stylesheet" href="styles/mystyle.css")
  • You can change the width, the style, and the color of a border using these 3 codes:
    (border-width: 10px;)
    (border-style: solid;)
    (border-color: #FFD700; /* Gold */) Like this border ->
  • You can make comments/notes to keep you organized in css using this:
    (/* Insert your text here! */)
  • You can change the style of the body using this code:
    body { all your code for css }
  • You can change the alignment of text using:
    (text-align: right,left,or center;)
  • A example of a layout of code in css for a block of content could look like:
    h1 {
    font-size: 2.5em;
    font-weight: bold;
    color: #000000; /* Gray */
    font-family: Arial Black;
    text-align: center;
    background-color: #FFD700;
    text-align: center;
    padding: 0;
    border: 3px solid black;
    border-style: solid;
    border-color: grey;
    margin-left: 10%;
    margin-right: 10%;
    }
